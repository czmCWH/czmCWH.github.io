hexo搭建博客
2020-06-15|工具

Hexo 是基于 Node.js 驱动的一款博客框架，它 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Github Page 是 Github 提供的一种免费的静态网页托管服务。所以使用 Github Page + Hexo 搭建个人博客。
