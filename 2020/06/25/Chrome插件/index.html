Chrome插件
2020-06-25|工具

Imagus

放大缩略图，当鼠标悬停显示链接中的图像/视频。

Octotree

看GitHub时，浏览器右侧显示项目的目录结构。

捕捉网页截图 - FireShot的

可以自动滚动截图，保存GitHub每日趋势图和自己仓库图等很有作用。

JSONView

格式化json，如果是get请求，返回的json数据会结构清晰的显示在网页上。

Dark Reader

网页的黑夜模式。将网页的白底变成黑底，适合长时间看博客文章。

React Developer Tools

React调试工具，可以当图标高亮显示时，说明该网页使用了 react
